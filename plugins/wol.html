Wake On Lan¶

Code author: Marcus Popp

This plugin awakes computers by a network message. A magic Wake-on-LAN packet.

Requirements¶

This plugin has no requirements or dependencies.

Configuration¶

plugin.conf¶

[wol]
    class_name = WOL
    class_path = plugins.wol

items.conf¶

[test]
    [[wol]]
        type = bool
        wol_mac = 03:06:a0:87:22:33
